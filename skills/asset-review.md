# Skill: 资产审核 (Asset Review)

## 审核规则
1. **禁止群体角色**：角色姓名中不得包含“众人”“大家”“宇航员们”“科学家们”“监测人员”等群体性词汇。若发现，标记为违规，要求拆分为个体。
2. **角色唯一性**：每个角色应有唯一 ID，且姓名不应重复（除非是别名）。
3. **地点覆盖**：所有在小说中出现的场景地点都应被提取，若缺少重要地点，标记为遗漏。
4. **道具准确性**：道具应是具体物件，不应是抽象概念。

## 修正指引
- 若发现群体角色，立即要求将其拆分为多个个体角色（如 CHAR_XXX_1, CHAR_XXX_2），并分别赋予简短描述。
- 若缺少地点，要求补充。

## 输出格式
JSON: { "pass": true/false, "issues": ["问题描述"] }
